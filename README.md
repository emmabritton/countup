# Countup

Counts up from a specified date

(this was built with a game engine so best not to leave it running)

## Usage

```
Usage: countup [OPTIONS]

Options:
  -d, --date <DATE>  Date to count from, format yyyy-mm-dd
  -h, --help         Print help
  -V, --version      Print version

```
